Feedback inline. Usar alertas de color para estados semánticos; el error block PEP/CPE es el único bloqueante de pantalla completa.

Tu DPI está próximo a vencer. Actualiza tu documento antes del 30 de junio.

El código ingresado no es válido. Verifica tu aplicación autenticadora.
FEIS — Cuenta simplificada
Esta cuenta tiene restricciones regulatorias. El monto máximo de inversión es Q 500,000 anuales.
Error block PEP/CPE (bloqueante)

⛔ Cuenta no disponible

De acuerdo con la regulación de la Superintendencia de Bancos, las cuentas FEIS no están disponibles para Personas Expuestas Políticamente (PEP) ni para personas clasificadas como CPE. Si consideras que esto es un error, contáctanos al +502 2222-3333.
