Tooltip de ayuda contextual

El ícono ⓘ aparece al lado de labels en inputs que requieren orientación. Al hacer hover o focus se muestra el tooltip con fondo oscuro. Se implementa con un span.help-icon y el atributo data-help.

Patrón — input con ayuda contextual
Q
Monto mínimo según producto seleccionado.
Estructura HTML
<div class="field">
  <label class="label">
    Nombre del campo
    <span class="help-icon" tabindex="0" data-help="Texto de ayuda aquí">i</span>
  </label>
  <input class="input" type="text" />
</div>
Reglas de uso
  • El tooltip se activa con hover y también con focus (accesibilidad por teclado via tabindex="0").
  • El atributo data-help contiene el texto del tooltip. Máximo 220px de ancho; el texto hace wrap automático.
  • Aplicar al menos en: campos de monto (mínimos/máximos), porcentaje de beneficiarios, fecha de vencimiento y referencia bancaria.
  • El tooltip aparece sobre el ícono para evitar solaparse con el input debajo.
  • No usar el ícono en campos autoexplicativos (nombre, correo, etc.).
