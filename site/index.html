Pdei+ Vanilla Components

Ejemplos de componentes

Construye pantallas del portal con clases listas para PHP y Livewire.

Esta documentación usa el mismo `lib/` que se entrega al equipo. Cada ejemplo muestra variaciones, código HTML y estilos auxiliares cuando aplican.

Getting started

Instalación

Copia `lib/` en tu proyecto y carga los tres archivos base.

Imports mínimos

<link rel="stylesheet" href="/lib/pdei.min.css">
<link rel="stylesheet" href="/lib/icons/pdei-icons.css">
<script src="/lib/pdei.min.js" defer></script>
Node/npm solo se usa para generar el bundle.
El frontend productivo consume archivos estáticos desde /lib.
